📲 홈 화면에 BuildNote 설치

BuildNote를 스마트폰 홈 화면에 설치하면 일반 앱처럼 바로 실행할 수 있습니다.

설치 가능 여부를 확인하고 있습니다.
Android · Chrome에서 수동 설치
  1. Chrome 오른쪽 위 메뉴(⋮)를 누릅니다.
  2. ‘앱 설치’ 또는 ‘홈 화면에 추가’를 선택합니다.
  3. 설치를 누르면 홈 화면에 BuildNote가 생깁니다.
iPhone · Safari에서 설치
  1. 이 페이지를 Safari로 엽니다.
  2. 화면 아래 공유 버튼(□↑)을 누릅니다.
  3. ‘홈 화면에 추가’ → ‘추가’를 누릅니다.
설치 버튼이 활성화되지 않을 때

주소를 복사해 Android는 Chrome, iPhone은 Safari에서 열어주세요. 설치 기능은 보안 연결(HTTPS)에서 제공됩니다.

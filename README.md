# AIMAX002 — 중장비 관리 (차계부 형식)

서버 없이 브라우저에만 저장되는 정적 PWA입니다. 데이터는 사용하는 스마트폰(브라우저)의 `localStorage`에 저장되며, 서버/DB가 필요 없습니다.

## 화면 구성

- **홈**: 선택한 장비의 이번달 유지비, 현재 계기판, 이번달 주행거리, 평균연비와 정비·주유 통합 내역(통장식)을 보여줍니다.
- **장비**: 장비 등록/삭제, 데이터 내보내기(백업)·가져오기(복원)
- **가동**: 날짜별 계기판 누적거리 기록 (직전 기록 대비 증가분 자동 표시)
- **정비**: 정비 내역/비용 기록
- **주유**: 주유 기록 + 평균연비(km/L) 자동 계산

## 데이터 보관 주의사항

모든 데이터는 접속한 기기의 브라우저에만 저장됩니다. 브라우저 데이터(사이트 데이터)를 지우면 기록이 함께 사라지므로, **장비 탭의 "내보내기" 버튼으로 주기적으로 JSON 백업**을 받아두는 것을 권장합니다. 백업 파일은 "가져오기" 버튼으로 복원할 수 있습니다.

## Vercel 배포

이 저장소는 순수 정적 파일(`index.html`, `manifest.json`, `sw.js`, `static/`)로만 구성되어 있어 빌드 과정 없이 바로 배포됩니다.

- Repository: `lycian5/AIMAX002`
- Project Name: `aimax-002` (URL: `https://aimax-002.vercel.app`)
- Framework Preset: Other
- Build/Install Command: 비활성화 (`vercel.json`에서 처리)
- Output Directory: `.` (저장소 루트)

배포 후 스마트폰 브라우저로 `https://aimax-002.vercel.app` 접속 → 공유 메뉴에서 "홈 화면에 추가"를 하면 앱처럼 바로 실행할 수 있습니다(PWA, 오프라인 지원).

## 로컬 실행

```bash
python -m http.server 8080
```

브라우저에서 `http://127.0.0.1:8080` 접속.
